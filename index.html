Strategic classification: the impact of user information and non-linear classification models

Machine learning models are widely used in high-stakes decisions, but their assumptions often fail when users adapt their features to achieve better outcomesâ€”a behavior known as gaming. While strategic classification addresses this distribution shift, most research focuses on linear models and assumes users have full knowledge of the system, which is often unrealistic. In my thesis I examine how non-linearity and varying levels of information affect gaming behavior. Instead of analyzing only extreme cases (full transparency vs. no information), I explores the impact of providing model explanations as well. Findings show that non-linear models are in many cases benefitial for both users and decision-makers, while different information scenarios create trade-offs: less information increases user costs, whereas transparency can disadvantage decision-makers.

CNN interpretability workflow with Sparse Autoencoders

Deep learning models excel in computer vision but often lack interpretability. Improving their transparency can enhance trust, error analysis, and bias mitigation. This project developed a workflow to better understand model internals by disentangling features in superpositions and visualizing them. A modified ResNet-18 was trained for image classification, then analyzed using feature visualizations both directly and with a sparse autoencoder. The sparse autoencoder revealed more interpretable features compared to direct visualization.

How do ML models influence us?

In this project, I explored the framework of strategic classification, where machine learning models are leveraged by decision-makers to guide user-related decisions. However, when users perceive the outcomes of these decisions as important, they may attempt to manipulate their features to achieve a more favorable result. This behavior is known as gaming. The goal of my project was to examine how different machine learning models and cost functions influence user behavior. To achieve this, I compared the optimal feature changes made by users in simulations across multiple models and datasets. I visualized the effects of these changes, providing insights into the interaction between user behavior and model decisions.

Covid-19 visualization dashboard with D3

This project aimed to create an interactive dashboard for two users using the Covid-19 dataset. The dashboard was designed to explore relationships between virus spread and vaccinations, compare vaccination trends, identify correlations with death rates, and find outliers in death rates across countries. It features a centralized map, flexible scatterplots, a clear juxtaposition of line charts, and multiple interactive components such as sliders and buttons. It was designed and programmed from scratch using D3 to provide a versatile and intuitive experience for users.
